GymFit es una aplicación web orientada al seguimiento de actividades fitness, que permite a los usuarios registrarse, iniciar sesión y gestionar su cuenta, incluyendo cambio de contraseña. El sistema muestra información de la membresía, así como apartados de rutinas y notificaciones dentro de una interfaz organizada.
